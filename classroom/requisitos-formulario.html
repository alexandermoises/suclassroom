Requisitos internos del administrador

Plataforma e-learning para emprendedores.

Esta sección no es pública. Solo debe verla el administrador después de iniciar sesión para revisar los requisitos internos del sistema.

Flujo principal

Flujo que debe cumplir el sistema

La primera versión debe validar el núcleo del modelo: aprender, aplicar, entregar, recibir feedback, mejorar y avanzar.

1

Registro obligatorio

El usuario crea cuenta por rol: alumno, mentor, administrador o empresa/ONG. Los requisitos internos solo son visibles para administrador.

2

Diagnóstico inicial

El alumno indica si tiene emprendimiento, empresa, programa social o idea de proyecto.

3

Curso inicial gratis

Accede al primer curso gratuito con módulos, videos, guías y plantillas.

4

Entregable obligatorio

Entrega formularios, plantillas, PDF, presentación o enlace con su avance real.

5

Evaluación del mentor

El mentor aprueba, rechaza o devuelve feedback accionable para corregir.

6

Pago y asesoría

Después de enviar el proyecto, el pago habilita revisión + asesoría y desbloquea el siguiente curso.

Roles del sistema

Cada tipo de usuario tiene funciones diferentes

Estos roles salen del formulario y están reflejados en el registro, login y paneles.

Alumno

Accede a cursos, completa módulos, desarrolla entregables, envía proyectos, recibe feedback, agenda asesorías, revisa progreso y accede a recursos.

Ver panel alumno

Mentor

Revisa proyectos, da retroalimentación estructurada, aprueba o rechaza entregables, atiende asesorías y da seguimiento.

Ver panel mentor

Administrador

Gestiona usuarios, cursos, contenidos, criterios de evaluación, mentores, pagos, accesos, reportes y calidad del servicio.

Ver panel admin

Empresa / ONG

Registra grupos, invita participantes, asigna cursos, monitorea avance, solicita programas y revisa reportes institucionales.

Ver panel empresa
Funcionalidades indispensables

Lo que debe tener SUClassroom

Estas funcionalidades aseguran que el usuario no solo aprenda, sino que ejecute y avance en su proyecto de forma estructurada.

Cursos progresivos

Acceso desbloqueable según avance, entregables y proyecto aprobado.

Módulos estructurados

Videos, guías, plantillas, lecturas, ejemplos y casos reales.

Gestión de proyectos

Desarrollar, guardar, actualizar, corregir y reenviar entregables.

Evaluación

Criterios claros, validación por mentor y aprobación del proyecto.

Retroalimentación

Feedback escrito y asesorías accionables para mejorar el proyecto.

Mentorías

Agenda para reservar sesiones con expertos después del pago.

Dashboard

Avance, estado del proyecto, próximos pasos y métricas por usuario.

Roles diferenciados

Alumno, mentor, administrador y empresa/ONG con funciones propias.

Panel B2B

Empresas y ONGs monitorean grupos, cumplimiento e impacto.

Reportes

Finalización, proyectos entregados, aprobados, rechazados e ingresos.

Pagos

Pago o simulación con Culqi para asesorías, revisión o suscripción.

Recursos

Repositorio de plantillas y herramientas según avance del usuario.

Soporte

Canales: WhatsApp, chat y correo. La asesoría principal se realiza por videollamada en tiempo real.

Tipo de plataforma

Web y mobile. La primera versión puede integrar herramientas externas para video, agenda y almacenamiento.

Seguridad y acceso

Registro obligatorio, email y contraseña, Google/redes sociales, verificación de email y sin compartir cuentas.

Contenido del curso

Objetivo, resultados, proyecto, pasos, criterios, plantillas, recursos, tiempo estimado, nivel y categoría.
