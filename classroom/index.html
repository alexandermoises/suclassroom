Basado en el formulario

SUClassroom convierte aprendizaje en ejecución real.

La plataforma no solo muestra cursos: guía al alumno para aplicar, entregar, recibir feedback, pagar asesoría y avanzar a nuevos cursos. Los requisitos internos solo los puede revisar el administrador.

Flujo MVP

Aprender -> aplicar -> entregar proyecto -> recibir feedback -> mejorar -> pagar asesoría -> desbloquear el siguiente curso.

Funcionalidades indispensables

¿Qué funcionalidades considera indispensables para el sistema?

El sistema debe asegurar que el usuario no solo aprenda, sino que ejecute y avance en su proyecto de forma estructurada.

Estas funcionalidades son el requisito principal: cada usuario debe tener su cuenta por rol y la plataforma debe conectar cursos, proyectos, mentorías, pagos, reportes y recursos.

Cursos progresivos y desbloqueables

El acceso depende del avance del alumno y del cumplimiento de entregables.

Módulos de contenido estructurado

Videos, guías y plantillas enfocadas en aplicación práctica.

Gestión de proyectos del usuario

Envío y evaluación de proyectos

Permite validar proyectos con criterios claros y revisión por mentores.

Retroalimentación estructurada

Agenda de mentorías

Sistema de roles

Diferencia funciones para estudiantes, mentores, administradores, empresas y ONGs.
