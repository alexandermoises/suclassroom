Datos del alumno
Al crear la cuenta, el alumno entra a la primera clase gratis. Para acceder a asesoría con un mentor debe presentar su proyecto y realizar el pago.
Datos del mentor
El mentor tendrá una cuenta para revisar proyectos, dar retroalimentación y atender asesorías pagadas.
Datos del administrador
El administrador tiene su propia cuenta para gestionar usuarios, cursos, mentores, proyectos, reportes, pagos y requisitos internos.
Datos de empresa / ONG
La empresa/ONG entra a un panel institucional para monitorear participantes, avance, reportes de impacto y mentorías grupales.
